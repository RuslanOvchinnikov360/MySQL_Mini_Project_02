# SQL Mini Project 2

## Project Overview
This project demonstrates practical SQL skills using MySQL. The work included database creation, table creation, data loading, data analysis, aggregation, filtering, subqueries, and joins.

## Tasks Completed
### Part 1: User Activity Analysis
Created a MySQL database and analyzed user activity data from the `users` table.
Completed tasks:
* Calculated the number of unique users for a selected date range.
* Identified the user with the highest total number of advertisement views.
* Found the day with the highest average number of viewed ads per user, considering only days with more than 500 unique users.
* Calculated user lifetime on the platform based on active days.
* Identified the user with the highest average daily ad views among users active for at least 5 days.

## Part 2: Product and Employee Sales Analysis
Created a separate MySQL database with two tables:
* `T_TAB1` - product sales data
* `T_TAB2` - employee data

Processed data included:
* 10 product sales records
* 5 employees
* 4 product categories
* 4 active sellers
* 1 employee with no sales

Completed tasks:
* Found unique product categories.
* Calculated total quantity and amount for mobile phone sales.
* Selected employees with salary above 100,000.
* Calculated minimum and maximum age and salary.
* Calculated average quantity sold for keyboards and printers.
* Calculated total sales amount by seller.
* Joined sales and employee tables to analyze seller details.
* Identified employees with no sales.
* Filtered employees by age.
* Checked JOIN behavior for employees without matching sales records.
